Параметры панели спавна объектов

Default — стандартный режим: небольшие объекты (например, кружки) появляются там, куда ты кликаешь, без привязки, а крупные, вроде стен, автоматически выравниваются по сетке.

PlaceFree — полностью свободное размещение, без какой-либо привязки к сетке или позициям.

PlaceNearby — ограничивает радиус появления объектов примерно до 2 тайлов от курсора.

SnapgridCenter — выравнивает объект по центру выбранного тайла.

SnapgridBorder — выравнивает объект по границе тайла.

Параметры выравнивания по сетке

AlignSimilar — попытка привязаться к подобным объектам. Сейчас не работает.

AlignTileAny — строгое выравнивание по центру любого тайла.

AlignTileEmpty — размещение только в пустых тайлах.

AlignTileNonDense — целевой тайл не должен содержать плотных объектов (с коллизией).

AlignTileDense — наоборот, объект появится только на тайле с плотным объектом.

AlignWall — привязка к вертикальным полутайлам (обычно используется для стен).

AlignWallProper — выравнивает объект по центру краёв тайла, идеален для ровного размещения стен и перегородок.
